# 14-day launch calendar

Assumes **Day 0 = public launch day**. Adjust dates as needed.

| Day     | Focus       | Actions                               |
| ------- | ----------- | ------------------------------------- |
| **-7**  | Capture     | Screenshots + raw trailer clips       |
| **-6**  | Edit        | 15s + 30s trailers; OG image          |
| **-5**  | Copy freeze | Final social / PH / email             |
| **-4**  | Soft QA     | 5 external playtests                  |
| **-3**  | Fix         | P0 bugs only                          |
| **-2**  | Assets      | Upload gallery drafts; schedule posts |
| **-1**  | Quiet       | Final smoke; sleep                    |
| **0**   | Launch      | Checklist T0; stay online             |
| **+1**  | Engage      | Reply all; clip UGC                   |
| **+2**  | Devlog      | Short “what shipped” post             |
| **+3**  | Community   | Poll next feature; follow-up email    |
| **+5**  | Content     | 90s YouTube explainer live            |
| **+7**  | Patch       | Small polish patch + notes            |
| **+14** | Review      | Metrics retro; plan v1.1 moment       |

## Content cadence (post-launch evergreen)

- **Weekly:** 1 clip + 1 dev note
- **Biweekly:** 1 feature deep-dive
- **Monthly:** larger “district update” launch moment
