## Description (first 2 lines matter most)

```
IronHaven is a free cyberpunk multiplayer game that runs entirely in your browser — no download, no launcher.

Create a street operator (Runner, Enforcer, Ghost, Fixer), customize modular gear, and drop into a shared neon city with live players, combat, vehicles, factions, and district events. Optional IronHaven Pass ($1.99/week) supports development; core play stays free.

Built with React, TypeScript, Three.js, Rapier, Supabase, and Stripe.
```
