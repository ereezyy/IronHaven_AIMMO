## 100 words

IronHaven AIMMO drops players into a neon metropolis built for the open web. There is no launcher and no install — open the site, create an operator (Runner, Enforcer, Ghost, or Fixer), and join a multiplayer city powered by Supabase realtime sync. Systems include combat, vehicles, police heat, economy, shops, missions, district factions, and dynamic world events. Appearance and modular avatar parts replicate to other players. An optional IronHaven Pass ($1.99/week) supports ongoing development while the core loop stays free. Built with React, TypeScript, Three.js, and Rapier.
