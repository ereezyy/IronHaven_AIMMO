# IronHaven — Brand Guidelines

## Name

| Form                | When                                       |
| ------------------- | ------------------------------------------ |
| **IronHaven AIMMO** | Formal / first mention / press titles      |
| **IronHaven**       | Short form, UI, social after first mention |
| **IH**              | Internal only — not for public marketing   |

**Avoid:** Ironhaven (lowercase H in middle of sentence is OK for “the ironhaven docks” flavor; product name keeps capital H), Iron Haven AIMMO as logo lockup, “Iron Haven Online.”

## Logo & marks

| Asset      | Path                                          | Notes                  |
| ---------- | --------------------------------------------- | ---------------------- |
| Logo PNG   | `media/assets/brand/logo.png`                 | Primary mark           |
| Logo SVG   | `media/assets/brand/logo.svg`                 | Vector where available |
| Banner     | `media/assets/brand/banner.png`               | Wide hero / GitHub     |
| Title card | `media/assets/brand/ironhaven_title.png`      | Cinematic title        |
| City intro | `media/assets/brand/ironhaven_city_intro.png` | Environment hero       |
| Beacon     | `media/assets/brand/ironhaven_beacon.png`     | Iconic prop / symbol   |

### Logo rules

1. Prefer **dark backgrounds** (`#08080a` or near-black).
2. Keep clear space ≈ ¼ logo height on all sides.
3. Do not recolor the mark into neon rainbow gradients.
4. Do not add drop shadows that wash out the red accent.
5. Minimum digital width: **128px** for recognition.

## Color system

| Role                      | Hex       | Usage                              |
| ------------------------- | --------- | ---------------------------------- |
| **Iron red**              | `#c03a30` | Primary accent, CTAs, faction heat |
| **Brass gold**            | `#c9a15a` | Premium, Pass, rewards             |
| **Void black**            | `#08080a` | Panels, backgrounds                |
| **Edge**                  | `#222428` | Borders, HUD chrome                |
| **Text**                  | `#e5e5e8` | Primary copy                       |
| **Dim**                   | `#9ca3af` | Secondary copy                     |
| **Muted**                 | `#5a5d62` | Labels, disabled                   |
| **Neon lime** (faction)   | `#39ff14` | Dock Rats only — sparingly         |
| **Chrome blue** (faction) | `#3f7d9a` | Chrome Guard only — sparingly      |

### Typography feel

- **UI / HUD:** monospaced or near-mono (tracking wide on labels)
- **Marketing headers:** condensed industrial sans or sharp grotesque
- **Body:** clean sans, high contrast on dark

Marketing rule: **ALL CAPS micro-labels** are on-brand for HUD; full sentences should use normal casing for readability.

## Voice

| Do                                                       | Don't                                            |
| -------------------------------------------------------- | ------------------------------------------------ |
| Concrete verbs: jack in, run the district, light them up | Empty superlatives: revolutionary, game-changing |
| Street confidence                                        | Cringe edgelord for its own sake                 |
| Honest early-live framing                                | Fake “millions of players”                       |
| Short CTAs: **Play free** / **Jack in**                  | Corporate: “Get started with our platform”       |

### Signature phrases (approved)

- Zero install. Full neon.
- Your streets.
- Jack in.
- Build your operator.
- The city does not sleep. Neither do the warrants.

---

## Photography / capture style

- Prefer **night / wet streets / neon bounce**
- Slight film grain OK; avoid oversaturated “fortnite brights”
- UI chrome can stay in frame if readable
- Always include the URL or “Play free in browser” on paid social creatives

---

## Pass branding

- Name: **IronHaven Pass**
- Price display: **$1.99/week** (always include period)
- Never imply Pass is required to play
- Visual: gold accent (`#c9a15a`) on void black
