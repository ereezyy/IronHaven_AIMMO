# IronHaven AIMMO — Media & Release Package

**Version:** 1.0.0  
**Status:** Public launch pack  
**Play now:** https://ironhaven-aimmo.vercel.app  
**GitHub:** https://github.com/ereezyy/IronHaven_AIMMO  
**Package root:** `/media`

This folder is the single source of truth for launch marketing, press, social, video, and brand assets.

---

## Quick start (ship in 60 minutes)

1. Read **[RELEASE_NOTES_v1.0.0.md](./RELEASE_NOTES_v1.0.0.md)** — what you're shipping.
2. Follow **[launch/LAUNCH_CHECKLIST.md](./launch/LAUNCH_CHECKLIST.md)** — day-of sequence.
3. Grab ready-to-post copy from **[copy/](./copy/)**.
4. Capture remaining screenshots using **[visual/SCREENSHOT_CHECKLIST.md](./visual/SCREENSHOT_CHECKLIST.md)**.
5. Record trailer clips from **[video/CAPTURE_SHOT_LIST.md](./video/CAPTURE_SHOT_LIST.md)**.

## Positioning (one sentence)

**IronHaven** is a free-to-play, browser-native cyberpunk MMO where you build a street operator, fight for districts, and run with real players — no download, no launcher, just the link.

---

## Voice & tone

- **Street-confident**, not corporate.
- **Concrete** (features, systems, streets) over hype words.
- **Honest** about early access / live systems.
- Avoid: “revolutionary,” “disrupt,” Web3-unless-true, fake player counts.
