# Screenshot checklist & quality bar

## Quality bar

- [ ] 1920×1080 or higher
- [ ] No desktop taskbar
- [ ] No personal Discord/email overlays
- [ ] Neon readable, not pure black crush
- [ ] UI text legible at 50% scale
- [ ] Consistent night-cyber look across set
